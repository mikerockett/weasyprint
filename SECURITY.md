# Security Policy

## Supported Versions

See [weasyprint.rockett.pw/versions.html](https://weasyprint.rockett.pw/versions.html) for the current support matrix.

## Reporting a Vulnerability

If you discover a security vulnerability in this package, please report it privately via email to **os@rockett.pw**. Do not open a public issue.

You should receive an acknowledgement within 48 hours. A fix will be developed privately and released as a patch version as soon as it is ready.

Please include:

- A description of the vulnerability
- Steps to reproduce or a proof of concept
- The version(s) affected
